## ğŸ“– Learning Philosophy

### The Three Questions
Before every mission, ask yourself:
1. **What am I building?** (Architecture)
2. **How do the pieces connect?** (Integration)
3. **What could go wrong?** (Error handling)

### Working with AI
- **Don't ask:** "Write me a login system"
- **Do ask:** "I'm building a login system using JWT tokens. First, help me design the user schema and authentication flow."

### When You Get Stuck
1. Read the error message carefully
2. Check browser console / server logs
3. Ask AI to explain the specific error
4. Google the exact error message
5. Take a break and come back fresh
